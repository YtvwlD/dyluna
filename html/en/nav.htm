About REPLACE_GAME_NAME

REPLACE_GAME_NAME is - or will be - a b rowser game - completely without Flash!

Instead, AJAX, JavaScript and Python are used.
The login is done via OpenID, a dezentral and open authentifcation model.

The graphical display is done by Bootstrap.

Font Awesome is used, too.
